About Me

I'm a self-taught programmer with a background in marketing. My passion for software development started when I built my first portfolio for a capstone course during my final semester at UNC-Chapel Hill. After that experience, I enrolled in programming courses and have spent the past few years learning JavaScript, HTML, CSS, and Python through free online resources and hands-on projects. One of my favorite learning experiences was Stanford Code in Place, a free program based on the first half of Stanford's CS106A (Intro to Python). There, I built fun projects using the Karel IDE, including a Quizlet-style program and a color generator.
